# sonic-pi README

Play Sonic Pi with your favorite editor!

## Features

You can play Sonic Pi from VSCode.

- Send codes to Sonic Pi and run it.
- Stop playing codes.

## Requirements

- Latest version of Sonic Pi

## Usage

1. Launch a Sonic Pi
2. Write some Code
3. Open Command Parrete, and call "Run code on Sonic Pi"

## Known Issues
